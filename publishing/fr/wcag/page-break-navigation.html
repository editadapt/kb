About

Version: WCAG 2.2+
Number: 2.4.13
Level: A
Applicability:
  • Page Breaks
  • Page List
Requirement:

For web content with page break locators, a mechanism is available to navigate to each locator.
Guidance:
  • Understanding Success Criterion 2.4.13
  • How to meet Success Criterion 2.4.13

Purpose

The purpose of this success criterion is to ensure that users have a way to access any static page break markers in a publication.

How to Meet

If there are page break markers in a publication, authors must provide a way for users to get to the page break locations.

For EPUB, providing a page list in the navigation document (or NCX in EPUB 2) is sufficient to meet this requirement.
